Classification Metrics
Project Details

ROC AUC


Definition:
ROC: Receiver operating characteristic. It's a metric used to evaluate classification models.

What I Did

- Built binary classification models using scikit-learn.
- Tested models on a synthetic fraud dataset with imbalance (90% legit / 10% fraud).
- Compared metric results across decision thresholds.
- Visualized confusion matrix, ROC curves, and precision-recall curves.

Technologies Used

  • Python (scikit-learn, matplotlib, seaborn)
  • Jupyter Notebook
  • Pandas & NumPy
