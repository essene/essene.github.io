LLM Basics

Notes and studies about AI Agents and Large Language Models.

What is an AI Agent?

An AI agent is a system capable of reasoning, planning and interacting with external tools in order to complete tasks autonomously.

Unlike traditional LLM interactions, agents can observe results, make decisions and execute multiple actions to achieve a goal.

Core Components of an Agent

  • LLM: Responsible for reasoning and generating responses.
  • Memory: Stores previous interactions and context.
  • Tools: External APIs, databases or functions the agent can use.
  • Planning: Defines the sequence of actions needed to solve a task.
  • Observations: Information collected after executing actions.

Common Challenges

  • Hallucinations
  • Tool selection errors
  • Context limitations
  • Latency and cost
